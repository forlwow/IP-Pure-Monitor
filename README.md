# 🛡️ IP纯净度实时监测脚本 (IP Pure Monitor)

这是一个用于 Tampermonkey (油猴) 的浏览器扩展脚本。它会在网页右下角生成一个美观的悬浮窗，实时显示你当前的网络 IP 地址、归属地、ISP 以及**欺诈分数（Fraud Score）**。

对于需要频繁使用境外服务（如 ChatGPT、Netflix、跨境电商等）并高度依赖 IP 纯净度的用户，此脚本可以帮助你随时掌握当前代理节点的健康状态。

基于`my.ippure.com` api 实现

内置请求缓存机制，频繁打开新网页不会滥发 API 请求。点击监控面板的**内容区域**，即可无视缓存，立即重新检测当前 IP。

需要修改请求时间请修改`const CACHE_MINUTES = 5;`
